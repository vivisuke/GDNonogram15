# 問題の著作権

一般的に、パズルの問題には著作権が自動的に発生します。

ですので、本プロジェクト以外への各問題の流用、各問題作者名・タイトルの削除・変更、および問題の修正は作者の許可が必要です。
それを行わない場合は著作権法違反となりますので、ご注意ください。
