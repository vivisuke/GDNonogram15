一般的に、パズルの問題には著作権が自動的に発生します。

ですので、本プロジェクトの各問題を流用する場合、各問題作者名・タイトルの削除・変更、および問題の修正は作者の許可が必要です。
それを行わない場合は著作権法違反となりますので、ご注意ください。
